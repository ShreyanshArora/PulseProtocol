# PulseProtocol

PulseProtocol is a SwiftUI-based, haptic-first Morse code game that lets players decode messages using **touch instead of sight**.

Rather than relying on visuals, PulseProtocol communicates information through short and long vibration patterns, encouraging players to understand rhythm, timing, and memory using haptics alone.
